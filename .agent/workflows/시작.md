---
description: 일자별 작업 시작 시 동기화 확인 및 환경 준비
---

이 워크플로우는 사용자가 작업을 시작할 때 다른 컴퓨터에서의 변경 사항을 안전하게 가져오기 위해 사용됩니다.

1. `.agent/last_sync_date.txt` 파일의 날짜를 확인하여 오늘 이미 체크했는지 검토합니다.
2. 날짜가 오늘이 아니거나 파일이 없다면:
   - `git fetch origin main`을 실행하여 원격 변경 사항을 확인합니다.
   - 로컬 브랜치가 뒤쳐져 있다면(`behind`) 사용자에게 알려주고 `git pull`을 제안합니다.
   - 사용자가 승인하면 `git pull origin main`을 실행합니다.
   - `.agent/last_sync_date.txt` 파일에 오늘 날짜를 기록합니다.
3. 모든 확인이 끝나면 `npm run dev`를 실행하여 개발 환경을 띄웁니다.
